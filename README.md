# emoji-js-code

> 用 emoji 写 js, [Demo.html](Demo.html) 是示例

| emoji |    对应 js    |
| :---: | :-----------: |
|  #️⃣   |      let      |
|  👉   |       =       |
|  🔚   |       ;       |
|  ▶️   |      (`       |
|  ◀️   |      `)       |
|  📢   |  console.log  |
|  ⚠️   | console.warn  |
|  🈲   | console.error |

## 特殊的

⏪x⏩ ---> ${x}
